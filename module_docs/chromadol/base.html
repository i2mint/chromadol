clear()[source]

This method is here, in fact, to disable the clear method, that would otherwise be inherited from MutableMapping. It’s existence is too dangerous, as it would delete all collections in the client. If you want to actually delete all collections in the client, do so explicitly by iterating over the client and deleting each collection, as such:

>>> for k in chroma_client_instance:
...     del chroma_client_instance[k]
